Descripción del proyecto

Este proyecto tiene como objetivo analizar el impacto de las nuevas tecnologías en la sociedad mediante el desarrollo de una solución informática enfocada en la protección básica de la información.

Para ello se diseñó e implementó un sistema de cifrado y descifrado basado en el algoritmo César, permitiendo comprender el funcionamiento general de los mecanismos de transformación de mensajes utilizados como base conceptual de la seguridad digital.

El proyecto integra procesos de análisis, diseño funcional, arquitectura de software y desarrollo del sistema.

Objetivo del sistema

Desarrollar una aplicación que permita cifrar y descifrar mensajes utilizando el algoritmo César, aplicando principios básicos de programación, diseño de software y organización del código.

Funcionalidades del sistema
Ingreso de mensajes por parte del usuario.
Selección del número de desplazamiento.
Cifrado del mensaje.
Descifrado del mensaje.
Validación de entradas.
Visualización del resultado.
